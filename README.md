# You Need
1. Installed Node Js
2. Postgres 14

# Installation
1. clone the project, using 
`git clone git@github.com:nelaneli/assessment-keda.git`
2. install packages `npm install`
3. open postgres, connect to `5432`
4. create database named: `parking`
5. start npm, using `npm start`
